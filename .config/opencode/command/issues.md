search through existing issues in the current project's upstream GitHub repo using the gh cli to find issues matching this query:

$ARGUMENTS

consider:

1. similar titles or descriptions
2. same error messages, symptoms or stack traces
3. related functionality or components
4. similar feature requests

list any matching issues with:

- issue number and title
- brief explanation of why it matches the query
- link to the issue

if there are no clear matches, say so.
